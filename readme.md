    Restaurant Management System
</h1>

Our Restaurant Management System is a robust solution designed specifically for the food service industry. It optimizes restaurant operations,from maintaining order history to updating food items. It has separate panel for customers to place their order which is built on masm32.

<h1 align="left">
    App Features
</h1>

### For Customers:

1. User-Friendly Ordering: Easy-to-navigate interface with clear menu categories, item descriptions, and prices.
2. Cart: Easy Add to Cart option for customers.
3. Bill: Get a well structured bill with total price printed on the bill.txt file.

### For Restaurants:

1. Order History : View past orders which is connected to order-history.txt file which are fetchedÂ dynamically.
2. Update Food Items : Restaurant Admins can easily update names and prices of the food items in the admin panel.

    Installation
</h1>
